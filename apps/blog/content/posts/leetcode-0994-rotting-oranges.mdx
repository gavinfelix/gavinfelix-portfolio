### 思路

这是典型的 **多源 BFS**（multi-source BFS）：

- 先遍历网格，把所有腐烂橘子（值为 `2`）的坐标入队，同时统计新鲜橘子数量 `fresh`
- 每一分钟，处理队列当前这一层（`size = queue.length`）：
  - 把这一层的腐烂橘子向四个方向扩散
  - 遇到新鲜橘子（`1`）就变腐烂（改为 `2`）、入队，并且 `fresh--`
- 当队列空或 `fresh === 0` 结束

最终：

- 若 `fresh === 0`，返回累计分钟数
- 否则说明有橘子永远不会腐烂，返回 `-1`

### 代码实现（TypeScript）

```ts
function orangesRotting(grid: number[][]): number {
  const rols: number = grid.length;
  const cols: number = grid[0].length;
  const queue: [number, number][] = [];
  let fresh: number = 0;
  let minutes: number = 0;

  for (let r = 0; r < rols; r++) {
    for (let c = 0; c < cols; c++) {
      if (grid[r][c] === 1) fresh++;
      if (grid[r][c] === 2) queue.push([r, c]);
    }
  }

  const dir = [
    [1, 0],
    [-1, 0],
    [0, 1],
    [0, -1],
  ];

  while (queue.length > 0 && fresh > 0) {
    const size = queue.length;

    for (let i = 0; i < size; i++) {
      const [r, c] = queue.shift()!;

      for (const [dr, dc] of dir) {
        const nr = r + dr;
        const nc = c + dc;

        if (nr >= 0 && nr < rols && nc >= 0 && nc < cols && grid[nr][nc] === 1) {
          queue.push([nr, nc]);
          fresh--;
          grid[nr][nc] = 2;
        }
      }
    }

    minutes++;
  }

  return fresh === 0 ? minutes : -1;
}
```
