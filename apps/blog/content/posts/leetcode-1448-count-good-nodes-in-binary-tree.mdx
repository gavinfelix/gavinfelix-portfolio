### 思路

从根到当前节点路径上，若当前节点值 **不小于** 路径上此前出现的最大值，则该节点为「好节点」。

用 DFS 向下遍历，同时维护路径上的 `maxVal`：

- 若 `node.val >= maxVal`，当前节点是好节点，`count += 1`，并更新 `maxVal = node.val`
- 递归左右子树，把累计的好节点数相加

根节点路径上此前没有更大约束，初始 `maxVal` 取 `-Infinity` 即可保证根一定被计入。

### 代码实现（TypeScript）

```ts
function goodNodes(root: TreeNode | null): number {
  function dfs(node: TreeNode | null, maxVal: number): number {
    if (node === null) return 0;

    let count = 0;

    if (node.val >= maxVal) {
      count = 1;
      maxVal = node.val;
    }

    count += dfs(node.left, maxVal);
    count += dfs(node.right, maxVal);

    return count;
  }

  return dfs(root, -Infinity);
}
```
