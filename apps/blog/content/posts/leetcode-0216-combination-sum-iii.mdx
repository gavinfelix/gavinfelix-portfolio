### 思路

使用回溯枚举组合：

- `path` 记录当前组合
- `sum` 记录当前组合和
- `start` 控制下一个数字起点，保证组合不重复且递增（每个数字只用一次）

终止条件：

- 当 `path.length === k` 时，如果 `sum === n` 就把当前组合加入答案

剪枝：

- 由于数字是递增尝试，如果 `sum + i > n`，后面的数只会更大，可以直接 `break`

### 代码实现（TypeScript）

```ts
function combinationSum3(k: number, n: number): number[][] {
  const res: number[][] = [];
  const path: number[] = [];

  function backtrack(start: number, sum: number) {
    if (path.length === k) {
      if (sum === n) res.push([...path]);
      return;
    }

    for (let i = start; i <= 9; i++) {
      if (sum + i > n) break;

      path.push(i);
      backtrack(i + 1, sum + i);
      path.pop();
    }
  }

  backtrack(1, 0);
  return res;
}
```
